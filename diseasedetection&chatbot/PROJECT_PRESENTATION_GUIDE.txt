PROJECT NAME: Plant AI - Plant Disease Detection & Farmer Chatbot

PROBLEM STATEMENT:
- Farmers often struggle to identify plant diseases early
- Getting expert advice is difficult, especially in rural areas
- Language barriers (many farmers prefer Tamil over English)
- Need for quick, accessible, and affordable solutions

OUR SOLUTION:
A web application that:
1. Detects plant diseases from photos using AI
2. Provides a smart chatbot for farming advice
3. Supports both English and Tamil languages
4. Works with voice input and output for easy use

================================================================================
            HOW PLANT DISEASE DETECTION WORKS
================================================================================

STEP-BY-STEP PROCESS:

STEP 1: USER UPLOADS IMAGE
- Farmer takes a photo of the plant leaf
- Uploads it through our website (drag & drop or click)
- Supported formats: JPEG, PNG, WebP
- Maximum size: 10MB

STEP 2: IMAGE VALIDATION
- System checks if it's a valid image
- Checks file size is within limit
- Prepares image for AI processing

STEP 3: SENDING TO AI MODEL
- Image is sent to HuggingFace's MobileNetV2 model
- This model is specifically trained on plant diseases
- It can identify 50+ different plant diseases

STEP 4: AI ANALYZES THE IMAGE
- The AI model examines the leaf patterns
- Looks for disease symptoms (spots, discoloration, etc.)
- Compares with thousands of disease images it was trained on
- Calculates confidence score (how sure it is)

STEP 5: GETTING RESULTS
- AI returns the disease name (e.g., "Tomato Late Blight")
- Returns confidence score (e.g., 95%)
- Our system adds treatment recommendations

STEP 6: SHOWING TO USER
- Disease name displayed clearly
- Confidence shown as percentage with progress bar
- Treatment recommendations listed as steps
- Results also sent to chatbot for follow-up questions

EXAMPLE OUTPUT:
---------------
Disease: Tomato - Late Blight
Confidence: 92%
Recommendations:
  1. Remove infected leaves immediately
  2. Apply copper-based fungicide
  3. Ensure proper plant spacing
  4. Avoid overhead watering
  5. Rotate crops annually

================================================================================
               HOW THE CHATBOT WORKS
================================================================================

STEP-BY-STEP PROCESS:

STEP 1: USER SENDS A MESSAGE
- Farmer types a question like "How to treat late blight?"
- Or speaks the question using voice input
- Message is sent to our server

STEP 2: LANGUAGE DETECTION
- System automatically detects if it's English or Tamil
- Uses the "langdetect" library
- Response will be in the same language

STEP 3: BUILDING THE CONTEXT
- System gathers:
  * The current question
  * Previous conversation history (last 10 messages)
  * Any detected disease information
- This helps give relevant answers

STEP 4: CREATING THE PROMPT
- System creates a special instruction for the AI:
  "You are an agricultural expert helping farmers..."
- Includes rules like:
  * Keep answers short (6-10 lines)
  * Use simple language
  * Give practical advice
  * Focus on organic solutions

STEP 5: SENDING TO AI (Mistral AI via OpenRouter)
- The question + context is sent to Mistral AI
- This is a powerful language model
- It understands farming and agriculture topics

STEP 6: STREAMING RESPONSE
- AI starts generating the answer
- Response is sent back word by word (streaming)
- User sees the answer appearing in real-time
- This feels more natural and faster

STEP 7: SMART COMPLETION
- System checks if the answer is complete
- If cut off mid-sentence, asks AI to continue
- Ensures every response ends properly

STEP 8: SAVING TO MEMORY
- Both question and answer are saved
- Used for context in future messages
- Helps chatbot remember the conversation

EXAMPLE CONVERSATION:
--------------------
User: "My tomato leaves have brown spots. What should I do?"

Chatbot: "Based on your description, your tomatoes may have early 
blight disease. Here's what you can do:

1. Remove the affected leaves immediately
2. Apply neem oil spray every 7 days
3. Ensure plants have good air circulation
4. Water at the base, not on leaves
5. Add mulch to prevent soil splash

If symptoms continue, consider using a copper-based fungicide. 
Would you like more details about any of these steps?"

================================================================================
          HOW TEXT AND SPEECH INPUT/OUTPUT WORKS
================================================================================

TEXT INPUT:
-----------
Step 1: User types message in the text box
Step 2: Press Enter or click Send button
Step 3: Message goes to chatbot service
Step 4: Response appears in chat

SPEECH INPUT (Voice Recording):
-------------------------------
Step 1: User clicks the microphone button
Step 2: Browser asks for microphone permission
Step 3: User speaks their question
Step 4: Audio is recorded (up to 60 seconds)
Step 5: User clicks Stop to finish recording
Step 6: Audio file is sent to server
Step 7: Server converts speech to text using Google Speech Recognition
Step 8: Converted text is sent to chatbot as a message
Step 9: Response appears in chat

How Speech-to-Text Works:
- Audio is converted to WAV format
- Sent to Google's Speech Recognition API
- Supports both English and Tamil
- Auto-detects the language spoken

TEXT OUTPUT:
------------
Step 1: Chatbot generates response
Step 2: Text appears in the chat window
Step 3: Response streams in real-time (word by word)
Step 4: Full response visible when complete

SPEECH OUTPUT (Text-to-Speech):
-------------------------------
Step 1: User clicks the speaker icon on any message
Step 2: Request sent to server with the text
Step 3: Server uses Google Text-to-Speech (gTTS)
Step 4: Converts text to audio (MP3 format)
Step 5: Audio is sent back to browser
Step 6: Audio plays automatically
Step 7: User can stop playback anytime

How Text-to-Speech Works:
- Using Google TTS library (gTTS)
- Supports English voice ("en")
- Supports Tamil voice ("ta")
- Generates natural-sounding speech
